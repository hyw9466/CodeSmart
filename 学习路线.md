
# 🚀 FastAPI + AI Agent 学习路线

基于 CodeLens 项目的系统化学习计划，目标是掌握 FastAPI 框架和 AI Agent 开发。

---

## 📚 学习路线总览

| 周次 | 主题 | 核心目标 |
|------|------|----------|
| **第1周** | FastAPI 基础入门 | 掌握路由、请求、响应、依赖注入 |
| **第2周** | 异步编程与高级特性 | 掌握异步接口、SSE、文件操作 |
| **第3周** | AI Agent 核心概念 | 掌握 LLM、Embedding、RAG、LangChain |
| **第4周** | 项目架构深度剖析 | 理解项目各模块设计与实现 |
| **第5周** | 实战项目改造与扩展 | 独立完成功能扩展与部署 |
